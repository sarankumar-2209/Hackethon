🤖 Bot Detection Frontend
📌 Overview
The Bot Detection Frontend is a React-based web application designed to visualize bot detection results. It interacts with a backend API that analyzes user activity on social media to classify users as bots or genuine users.

🎯 Features
✔️ Real-time data visualization using Chart.js.
✔️ Dynamic UI with filters for bot detection results.
✔️ Interactive user insights and performance metrics.
✔️ Modular React components for easy scalability.

📂 Project Structure
🔹 Frontend (React)
src/

components/
Results.js – Displays bot detection results.
Chart.js – Implements bar chart visualization.
styles/
results.css – UI styling.
App.js – Main React app file.
index.js – Entry point.
public/

index.html – Base HTML file.
package.json – Project dependencies and scripts.

📊 Data Visualization Metrics
Metric- Description
Precision- Accuracy of bot classification
Recall- Ability to detect all bots
F1 Score- Balance between precision and recall
AUC-ROC- Classification performance metric
